KPI Analysis MVP

KPI app ที่อ่านความเคลื่อนไหวงาน แล้วแปลงเป็นคะแนนที่อธิบายได้

บันทึกงานรายวันหรือรายสัปดาห์ของคุณจะถูกสรุปเป็น evidence, mapped เข้าหัวข้อ KPI, ประเมินคะแนนเบื้องต้นด้วย AI และสามารถบันทึก history ขึ้น Firebase ได้

Provider: OpenAI Storage: Local mode Scoring: Evidence-based

Work Input

กรอก note ความเคลื่อนไหวงาน

Firebase: checking

AI จะช่วยประมวลผลให้ 4 อย่าง

  • สรุปสิ่งที่คุณทำเป็นภาษาที่อ่านง่าย
  • ดึง evidence และ impact ออกมาจาก note
  • map เข้า KPI แต่ละข้อ
  • เสนอคะแนน 1-5 พร้อมเหตุผลและคำแนะนำถัดไป
